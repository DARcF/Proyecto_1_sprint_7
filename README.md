# Proyecto Sprint 7 - Aplicación web con Streamlit

## Descripción
Esta aplicación web fue desarrollada con Streamlit para explorar un conjunto de datos de anuncios de venta de vehículos en Estados Unidos.

## Funcionalidad
La aplicación permite al usuario interactuar con los datos mediante dos visualizaciones:

- Un histograma para analizar la distribución del kilometraje (`odometer`).
- Un gráfico de dispersión para observar la relación entre el kilometraje (`odometer`) y el precio (`price`).

## Herramientas utilizadas
- Python
- pandas
- plotly
- streamlit

## Cómo ejecutar la aplicación
Desde la terminal, dentro de la carpeta del proyecto, ejecuta:

```bash
streamlit run app.py
